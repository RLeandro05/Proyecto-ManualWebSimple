Apache Tomcat en Ubuntu (Linux)

¿Qué es Apache Tomcat?

Apache Tomcat es un servidor web de código abierto que implementa las especificaciones de Java EE, pero de forma más sencilla que otros servidores más complejos. Es ampliamente utilizado para aplicaciones web basadas en Servlets y JSP ->[...].

Pasos para instalar Apache Tomcat en Ubuntu

  1. 1. Descargar Apache Tomcat

    Visita la página oficial de Apache Tomcat en Tomcat Downloads. Descarga el archivo .tar.gz de la última versión estable.

  2. 2. Instalación de Apache Tomcat

                                
# Navega al directorio de descargas:
cd /ruta/a/descargas

# Descomprime el archivo descargado:
tar xvf apache-tomcat-9.0.x.tar.gz

# Mueve el directorio a un lugar adecuado:
sudo mv apache-tomcat-9.0.x /opt/tomcat

  3. 3. Configuración de Apache Tomcat

    Edita el archivo server.xml para cambiar el puerto predeterminado: 

    
sudo nano /opt/tomcat/conf/server.xml

# Busca la línea y edita:
Connector port="8080" protocol="HTTP/1.1" ... /

  4. 4. Arrancar Apache Tomcat

    
# Para iniciar:
/opt/tomcat/bin/startup.sh

# Para detener:
/opt/tomcat/bin/shutdown.sh

  5. 5. Desplegar una aplicación

    Coloca tu archivo .war en el directorio webapps. Tomcat lo desplegará automáticamente.

Manual para Configurar Servidores Web para Ejecutar Java EE

¿Cuál es el objetivo de este manual?

Este manual tiene como objetivo explicar cómo configurar tres servidores web para ejecutar aplicaciones Java EE, incluyendo Servlets, JSP, WebSockets y EJBs, en sistemas operativos Windows y Linux. Se cubrirán tres servidores distintos que pueden ser utilizados en entornos de desarrollo web: Apache Tomcat, WildFly y GlassFish.

¿Qué es un servidor web?

Un servidor web es un software que utiliza los protocolos HTTP o HTTPS para servir contenido a los navegadores, permitiendo que los usuarios accedan a sitios y aplicaciones web. Este contenido puede incluir páginas HTML, imágenes, videos, y otros recursos.

¿Cómo funciona un servidor web?

Un servidor web escucha solicitudes de los clientes (navegadores) a través del protocolo HTTP. Responde con recursos solicitados, como documentos HTML, o aplicaciones para generar contenido dinámico.
